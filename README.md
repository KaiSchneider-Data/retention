# Portfolio 
Portfolio-project: Player Retention with SQL

## Objective

This project analyzes user retention and churn behavior 
using transactional data as a proxy for player behavior.

The goal is to understand how early user activity impacts retention.

## Business Question

- How many users return after their first activity?
- How does early engagement influence retention?
- Which users are most likely to churn early?

## Analysis Plan

1. Data exploration
2. Daily Active Users (DAU)
3. Retention (Day 1 and Day 7)
4. User segmentation
5. Churn analysis

## Next Steps

- Calculate Daily Active Users (DAU)
- Build Day 1 and Day 7 retention metrics
- Identify early churn patterns
